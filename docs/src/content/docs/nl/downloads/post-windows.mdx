## Windows SmartScreen

Windows waarschuwt gebruikers bij het installeren van software zonder certificaat.

![Smart screen](@assets/img/smart-screen.png)

Om de app te installeren, moet je op "Meer info" klikken.

![Slim scherm meer info](@assets/img/smart-screen-more-info.png)

Klik vervolgens op "Toch uitvoeren".
