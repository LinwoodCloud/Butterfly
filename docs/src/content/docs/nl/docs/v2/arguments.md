title: Argumenten

Je kunt de app starten met argumenten.

`vlinder.exe <Path>`

Pad is het pad naar het bestand dat je wilt openen.

## Instellingen

|          Optie | Beschrijving                                                                          |
| -------------: | :------------------------------------------------------------------------------------ |
| `--path`, `-p` | Kies de gegevensmap waar alle gegevens van Butterfly zijn opgeslagen. |
