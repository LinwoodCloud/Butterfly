title: حجج
---

يمكنك بدء التطبيق بالحجج.

`butterfly.exe <Path>`

المسار هو المسار إلى الملف الذي تريد فتحه.

## خيارات

|           خيار | الوصف                                                                 |
| -------------: | :-------------------------------------------------------------------- |
| `--path`, `-p` | اختر دليل البيانات حيث يتم تخزين جميع بيانات الفراشة. |
