Pour modifier l'arrière-plan de la page, cliquez sur l'icône <Wrench className="inline-icon"/> et là, sur l'icône <Book className="inline-icon"/> de la page. Là, vous aurez la possibilité de cliquer sur l'icône <Image className="inline-icon"/>. Sur un ordinateur, vous pouvez également utiliser le raccourci `ctrl-B`.

Lorsque vous configurez un backgournd, vous avez deux onglets. L'onglet <Globe className="inline-icon"/> Général pour sélectionner le type d'arrière-plan à utiliser, et le <Gear className="inline-icon"/> Pour configurer l'arrière-plan sélectionné.

## Schéma <Globe className="inline-icon"/>

Il y a 8 modèles de fond différents disponibles parmi lesquels vous pouvez choisir. Pour chacun des modèles disponibles, vous pouvez configurer les éléments suivants :

### Propriétés <Gear className="inline-icon"/>

Dans l'onglet de configuration horizontale, vous avez l'option suivante :

- Largeur - la largeur de chaque ligne respectivement.
- Compteur - vous pouvez regrouper les lignes de cette manière.
- Espace - la distance entre chaque groupe de lignes.

Par exemple, pour créer une feuille de musique, vous pouvez utiliser la configuration suivante:
`Width - 40`, `Count - 5`, `Space - 80`

La configuration verticale suit la même logique pour les colonnes au lieu des lignes

## <Globe className="inline-icon"/> Image

Le type d'arrière-plan de l'image vous permet de télécharger une image et de l'utiliser comme arrière-plan pour la page.

### Propriétés <Gear className="inline-icon"/>

L'échelle dans la direction x et y peut être modifiée si cette image est petite ou grande.
