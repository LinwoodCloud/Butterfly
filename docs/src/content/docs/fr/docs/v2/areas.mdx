## Introduction

![Area](area.png)

Les zones sont un moyen de limiter la taille de la toile. Vous pouvez créer des zones en utilisant [area tool](../tools/area). Si vous cliquez sur l'arrière-plan, vous pouvez ouvrir la boîte de dialogue de la liste des zones en utilisant le menu contextuel.

## Création et suppression des zones

Alors que vous pouvez directement modifier le contenu sur la toile, il est courant que vous vouliez restreindre la région dans laquelle vous pouvez modifier le contenu, ou que vous aimeriez pouvoir accéder rapidement à une partie du document. Pour cela, vous avez l'outil <Monitor className="inline-icon"/> Area . Comme expliqué dans la [section introduction](../), vous devrez peut-être ajouter le bouton <Monitor className="inline-icon"/> à votre barre d'outils. À l'aide de [area tool](../tools/area), vous pouvez dessiner un rectangle n'importe où sur la toile pour définir la zone.

## Plus de contrôle

Vous pouvez définir davantage l'outil zone en cliquant à nouveau sur le bouton <Monitor className="inline-icon"/>. Là, vous pouvez définir la largeur fixe, la hauteur ou le rapport d'aspect.

## Le menu contextuel de la zone

 Après avoir dessiné une zone, vous pouvez cliquer ou appuyer longuement sur la touche afficher la zone pour ouvrir un menu contextuel avec les options suivantes :

- `Name` - Donne à la zone un nom qui vous convaincra d'utiliser.
- `Entrez dans la région` - Vous permettant d'éditer du contenu uniquement bordé par cette zone.
- `Supprimer` - Supprime la zone. Ceci _ne supprime pas_ le contenu de cette zone
- `Export` - Exporte le contenu de la zone sélectionnée.
- `Ajouter au pack` - Ajoute la zone à un [pack](../pack)

## Une autre option

Une autre option pour vous d'utiliser le navigateur de zone est dans le menu de gauche, ou sur le menu papillon si vous utilisez une plateforme mobile.
