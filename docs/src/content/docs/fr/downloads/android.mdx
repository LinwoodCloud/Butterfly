## Le programme bêta sur le Play Store

Cliquez sur [ce lien](https://play.google.com/apps/testing/dev.linwood.butterfly) pour vous inscrire à la version [nightly](/community/nightly) de l'application. Veuillez noter que cette version peut avoir des bugs. La version Play Store sera automatiquement mise à jour lorsqu'une nouvelle version nocturne sera publiée.
