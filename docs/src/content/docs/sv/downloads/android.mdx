Klicka på [denna länk] (https://play.google.com/apps/testing/dev.linwood.butterfly) för att registrera dig för [nightly](/community/nightly) versionen av appen. Observera att denna version kan ha buggar. Play Store-versionen uppdateras automatiskt när en ny nattlig version släpps.
