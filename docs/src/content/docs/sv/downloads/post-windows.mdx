## Windows SmartScreen

Windows varnar användare när du installerar programvara utan ett certifikat.

![Smart skärm](@assets/img/smart-screen.png)

För att installera appen måste du klicka på "Mer info".

![Smart skärm mer info](@assets/img/smart-screen-more-info.png)

Klicka sedan på "Kör ändå".
