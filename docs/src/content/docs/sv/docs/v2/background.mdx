För att redigera bakgrunden på sidan, klicka på ikonen <Wrench className="inline-icon"/> och där, på ikonen <Book className="inline-icon"/> sida. Där har du möjlighet att klicka på ikonen <Image className="inline-icon"/>. På en dator kan du också använda `ctrl-B`-genvägen.

När du konfigurerar en backgournd, har du två flikar. Fliken <Globe className="inline-icon"/> Allmänt för att välja typ av bakgrund att använda och <Gear className="inline-icon"/> för att konfigurera den valda bakgrunden.

## <Globe className="inline-icon"/> Mönster

Det finns 8 olika bakgrundsmallar som du kan välja mellan. För vart och ett av de tillgängliga mönstren kan du konfigurera följande:

### <Gear className="inline-icon"/> egenskaper

I fliken horisontell konfiguration har du följande alternativ:

- Bredd - bredden för varje rad respektive.
- Räkna - du kan gruppera raderna i gruppen på det sättet.
- Utrymme - avståndet mellan varje grupp av rader.

Till exempel, för att skapa ett musikblad kan du använda följande konfiguration:
`Bredd - 40`, `Räkna - 5`, `Utrymme - 80`

Den vertikala konfigurationen följer samma logik för kolumner istället för rader

## <Globe className="inline-icon"/> Bild

Bakgrundstypen låter dig ladda upp en bild och använda den som bakgrund för sidan.

### <Gear className="inline-icon"/> egenskaper

Skalan i x och y riktning kan ändras om denna bild till liten eller stor.
