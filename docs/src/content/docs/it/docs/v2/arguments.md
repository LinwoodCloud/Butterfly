---
title: Argomenti
---

È possibile avviare l'applicazione con argomenti.

`butterfly.exe <Path>`

Il percorso è il percorso del file che si desidera aprire.

## Opzioni

|        Opzione | Descrizione                                                                                        |
| -------------: | :------------------------------------------------------------------------------------------------- |
| `--path`, `-p` | Scegliere la directory dei dati in cui tutti i dati di Butterfly sono memorizzati. |
