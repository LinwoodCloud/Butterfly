Για να επεξεργαστείτε το φόντο της σελίδας, κάντε κλικ στο εικονίδιο <Wrench className="inline-icon"/> και εκεί, στο εικονίδιο σελίδας <Book className="inline-icon"/>. Εκεί, θα έχετε την επιλογή να κάνετε κλικ στο εικονίδιο <Image className="inline-icon"/>. Σε έναν υπολογιστή, μπορείτε επίσης να χρησιμοποιήσετε τη συντόμευση `ctrl-B`.

Όταν ρυθμίζετε ένα backgournd, έχετε δύο καρτέλες. The <Globe className="inline-icon"/> General tab to select the type of background to use and the <Gear className="inline-icon"/> To configure the selected background. (Automatic Copy)

## <Globe className="inline-icon"/> Μοτίβο

Υπάρχουν 8 διαφορετικά πρότυπα φόντου που μπορείτε να διαλέξετε. Για κάθε ένα από τα διαθέσιμα μοτίβα, μπορείτε να ρυθμίσετε τα ακόλουθα:

### <Gear className="inline-icon"/> Ιδιότητες

Στην καρτέλα οριζόντιας διαμόρφωσης, έχετε την ακόλουθη επιλογή:

- Πλάτος - το πλάτος για κάθε γραμμή αντίστοιχα.
- Μετρητής - μπορείτε να ομαδοποιήσετε τις σειρές στην ομάδα με αυτόν τον τρόπο.
- Διάστημα - η απόσταση μεταξύ κάθε ομάδας των γραμμών.

Για παράδειγμα, για να δημιουργήσετε ένα μουσικό φύλλο, μπορείτε να χρησιμοποιήσετε τις ακόλουθες ρυθμίσεις:
`Width - 40`, `Count - 5`, `Space - 80`

Η κάθετη διαμόρφωση ακολουθεί την ίδια λογική για τις στήλες αντί για τις σειρές

## Εικόνα <Globe className="inline-icon"/>

Ο τύπος φόντου εικόνας σας επιτρέπει να ανεβάσετε μια εικόνα και να την χρησιμοποιήσετε ως φόντο για τη σελίδα.

### <Gear className="inline-icon"/> Ιδιότητες

Η κλίμακα στην κατεύθυνση x και y μπορεί να αλλάξει αν αυτή η εικόνα σε μικρή ή μεγάλη.
