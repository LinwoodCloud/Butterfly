:::tip[What apk should I use?]

**Universal** πρέπει να λειτουργεί σε όλες τις συσκευές. Αν θέλετε ένα μικρότερο apk, χρησιμοποιήστε αυτό που ταιριάζει με την αρχιτεκτονική της συσκευής σας.

:::

## Το πρόγραμμα beta στο play store

Κάντε κλικ στο [σε αυτόν τον σύνδεσμο](https://play.google.com/apps/testing/dev.linwood.butterfly) για να εγγραφείτε στην έκδοση [nightly](/community/nightly) της εφαρμογής. Παρακαλώ σημειώστε ότι αυτή η έκδοση μπορεί να έχει σφάλματα. Η έκδοση play store θα ενημερώνεται αυτόματα όταν κυκλοφορήσει μια νέα νυχτερινή έκδοση.
