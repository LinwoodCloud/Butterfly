## Έξυπνη Οθόνη Των Windows

Τα Windows προειδοποιούν τους χρήστες κατά την εγκατάσταση λογισμικού χωρίς πιστοποιητικό.

![Έξυπνη οθόνη](@assets/img/smart-screen.png)

Για να εγκαταστήσετε την εφαρμογή, πρέπει να κάνετε κλικ στο "Περισσότερες πληροφορίες".

![Έξυπνη οθόνη περισσότερες πληροφορίες: ](@assets/img/smart-screen-more-info.png)

Στη συνέχεια, κάντε κλικ στο "Run ούτως ή άλλως".
