## El programa beta en la tienda de juegos

Haz clic en [este enlace](https://play.google.com/apps/testing/dev.linwood.butterfly) para registrarte en la versión [nightly](/community/nightly) de la aplicación. Tenga en cuenta que esta versión puede tener errores. La versión Play Store se actualizará automáticamente cuando se libere una nueva versión nocturna.
