Para editar el fondo de la página, haga clic en el icono <Wrench className="inline-icon"/> y allí, en el icono <Book className="inline-icon"/>. Ahí tendrás la opción de hacer clic en el icono <Image className="inline-icon"/>. En una computadora, también puedes usar el acceso directo `ctrl-B`.

Al configurar un backgournd, tiene dos pestañas. La pestaña <Globe className="inline-icon"/> General para seleccionar el tipo de fondo a usar y la <Gear className="inline-icon"/> para configurar el fondo seleccionado.

## <Globe className="inline-icon"/> Pattern

Hay 8 diferentes plantillas de fondo disponibles entre las que puedes elegir. Para cada uno de los patrones disponibles, puede configurar lo siguiente:

### <Gear className="inline-icon"/> Propiedades

En la pestaña de configuración horizontal, tiene la siguiente opción:

- Ancho - el ancho de cada fila respectivamente.
- Cuenta - puede agrupar las filas en grupo de esa manera.
- Espacio - la distancia entre cada grupo de filas.

Por ejemplo, para crear una hoja de música, puedes usar la siguiente configuración:
`Width - 40`, `Count - 5`, `Space - 80`

La configuración vertical sigue la misma lógica para columnas en lugar de filas
