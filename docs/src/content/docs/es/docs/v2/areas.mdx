## Introducción

![Area](area.png)

Las áreas son una manera de restringir el tamaño de la tela. Puedes crear áreas usando la [herramienta de área](../tools/area). Si hace clic en el fondo puede abrir el diálogo de lista de áreas usando el menú contextual.

## Creando y eliminando áreas

Mientras que puedes editar directamente el contenido en el lienzo, es común que quieras restringir la región en la que puedes modificar el contenido, o que usted quiera que ot pueda acceder rápidamente a una parte del documento. Para eso, tienes la herramienta <Monitor className="inline-icon"/> Área. Como se explica en la [sección de introducción](../), es posible que tengas que añadir el botón <Monitor className="inline-icon"/> a tu barra de herramientas. Usando la [herramienta de área](../tools/area), puedes dibujar el rectángulo en cualquier lugar del lienzo para luego definir el área.

## Más control

Puede definir aún más la herramienta de área haciendo clic de nuevo en el botón <Monitor className="inline-icon"/> área. Ahí, puede definir la anchura fija, la altura, o la relación de aspecto.

## El menú contextual de área

 Después de dibujar un área, puede hacer clic o pulsar largo sobre el toque mostrar el área para abrir un menú contextual con las siguientes opciones:

- `Nombre` - Dale al área un nombre que sea conviniente para que lo utilices.
- `Entra en el área` - Te permite editar contenido sólo con el borde de esta área.
- `Eliminar` - Elimina el área. Esto _no_ elimina el contenido de esta área
- `Exportar` - Exporta el contenido del área seleccionada.
- `Añadir al paquete` - Agrega el área a un [pack](../pack)

## Otra opción

Otra opción para usar el navegador de área es en el menú de la izquierda, o en el menú de mariposa si está utilizando una plataforma móvil.
