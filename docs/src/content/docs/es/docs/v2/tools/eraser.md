With this tool you can erase the elements on the paper.

## Configuración

| ---------------------: | :---------: | :------------------------------------------------------------------------ |
|        Ancho del trazo |      5      | El ancho del trazo                                                        |
| Multiplicador de trazo |      1      | Si tienes un estilo, este es el multiplicador que se añadirá a la presión |
