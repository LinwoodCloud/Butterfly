---
title: 路径擦除工具
---

通过这个工具，您可以删除纸上的整个路径。

## 配置

|    财产 |  默认设置 | 描述                                    |
| ----: | :---: | :------------------------------------ |
|  笔画宽度 |   5   | 笔画的宽度                                 |
|  笔记倍数 |   1   | 如果您有一个样式表，这是将添加到压力中的乘数                |
| 包括擦除器 | false | 如果您可以删除擦除器的路径，此设置将被控制。 删除的擦除器下的路径将会显示 |
