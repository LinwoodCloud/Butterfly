要编辑页面的背景，请点击 <Wrench className="inline-icon"/> 图标和 <Book className="inline-icon"/> 页面图标。 在这里，您可以选择点击 <Image className="inline-icon"/> 图标。 在计算机上，您也可以使用 `ctrl-B` 快捷方式。

配置后端时，您有两个标签。 要选择要使用的背景类型的 <Globe className="inline-icon"/> 常规标签， <Gear className="inline-icon"/> 要配置选定的背景。

## <Globe className="inline-icon"/> 模式

您可以选择8个不同的背景模板。 对于每个可用模式，您可以配置以下内容：

### <Gear className="inline-icon"/> 属性

在水平配置选项卡中，您有以下选项：

- 宽度 - 各行的宽度。
- 计数 - 您可以按分组分组列出。
- 空格-每组行之间的距离。

例如，要创建一个音乐表，您可以使用以下配置：
`宽度- 40`, `Count - 5`, `Space - 80`

垂直配置遵循列而不是行相同的逻辑。

## <Globe className="inline-icon"/> 图像

图像背景类型允许您上传图像并将其用作页面的背景资料。

### <Gear className="inline-icon"/> 属性

x和y方向的缩放可以更改，如果这个图像大小或大小。
