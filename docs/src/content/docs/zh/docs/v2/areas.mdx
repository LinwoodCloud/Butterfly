## 一. 导言

![Area](area.png)

区域是限制画布面积的一种方式。 您可以使用[区域工具](../tools/area)创建区域。 如果您点击后台，您可以使用上下文菜单打开区域列表对话框。

## 创建和删除区域

当您可以直接编辑画布上的内容时， 常见的情况是，您想限制您可以修改内容的区域。 或者您想要的是能够快速访问部分文档。 为此，您有 <Monitor className="inline-icon"/> 区域工具。 正如[导言部分](../)中所解释的那样，您可能需要将 <Monitor className="inline-icon"/> 区域按钮添加到工具栏。 使用 [区域工具](../tools/area)，你可以绘制画布上任何地方的矩形，然后定义区域。

## 更多控制

您可以通过再次点击 <Monitor className="inline-icon"/> 区域按钮来进一步定义区域工具。 在这里，您可以定义区域的固定宽度、高度或宽度。

## 区域上下文菜单

 绘制区域后，您可以点击或长按触摸显示区域来打开上下文菜单，选择以下选项：

- `Name` - 给该区域一个令人信服的名称供你使用。
- `输入区域` - 允许您只能编辑此区域的内容。
- `Delete` - 删除区域。 此 _不_ 删除此区域内的内容
- `Export` - 导出选定区域的内容。
- "添加到包" - 将区域添加到一个 [pack](../pack)

## 另一个选项

您使用区域导航器的另一个选项在左菜单上。 如果您正在使用一个移动平台，或者在蝴蝶 菜单上。
