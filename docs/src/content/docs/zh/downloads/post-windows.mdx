## Windows SmartScreen

Windows 在安装没有证书的软件时警告用户。

![智能屏幕](@assets/img/smart-screen.png)

要安装应用程序，您需要点击"更多信息"。

![智能屏幕更多信息](@assets/img/smart-screen-more-info.png)

然后点击"无论如何运行"。
