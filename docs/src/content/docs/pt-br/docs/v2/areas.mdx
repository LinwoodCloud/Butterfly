## Introdução

![Area](area.png)

As áreas são uma forma de restringir a dimensão da tela. Você pode criar áreas usando a [ferramenta de área](../tools/area). Se você clicar em segundo plano você pode abrir a lista de áreas usando o menu de contexto.

## Criar e excluir áreas

Enquanto você pode editar diretamente o conteúdo da tela inteira, é comum querer restringir a área de edição ou definir uma área para acessa-la rapidamente.  Para isso, a ferramenta <Monitor className="inline-icon"/> de área está disponível. Como explicado na [seção de introdução](../), talvez você precise adicionar o botão <Monitor className="inline-icon"/> área à sua barra de ferramentas. Usando a [ferramenta de área](../tools/area), você pode desenhar retângulos em qualquer lugar da tela para definir uma área.

## Mais controle

Você pode definir a ferramenta de área clicando novamente no botão <Monitor className="inline-icon"/> área. Lá, você pode definir a largura, altura ou proporção de 
proporção de aspecto.

## Menu de contexto da área

 Depois de desenhar uma área, você pode clicar ou tocar e segurar na tela de toque para abrir um menu de contexto com as seguintes opções:

- `Nome` - Dê à área um nome que seja conveniente para você usar.
- `Entre na área` - Permite que você edite o conteúdo desta área.
- `Deletar` - Apague a área. Este _não_ exclui o conteúdo desta área
- `Exportar` - Exportar o conteúdo da área selecionada.
- `Adicionar ao pacote` - Adicione a área a um [pack](../pack)

## Outra opção

Outra opção para você utilizar o navegador de área está no menu à esquerda, ou no menu de borboleta se você estiver usando uma plataforma móvel.
