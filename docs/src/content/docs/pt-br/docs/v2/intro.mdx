> Bem-vindo à Borboleta, o aplicativo de notas de código-aberto.

A Borboleta é um aplicativo que grava anotações que busca uma experiência unificada em todas as plataformas. Ele fornece uma interface de usuário simples com personalização poderosa.

## Primeiros Passos

> Você pode usar a barra lateral para navegar rapidamente através da wiki.

Para começar a trabalhar no aplicativo, você tem duas opções:

1. [Downloads](/downloads) the app version that is appropriate for your platform (Butterfly can run on most mobile, desktop, and web platforms).
2. Use [versão web](https://web.butterfly.linwood.dev) se você não encontrar sua plataforma ou se você não quiser instalá-la nela. Veja [here](armazenamento#web) sobre o armazenamento de arquivos na versão web.

> Assim que você abrir o aplicativo, continue lendo.

## A visualização de documentos

Você pode retornar da visualização de documentos para a lista de documentos com a ação traseira do seu dispositivo. Como na visualização principal, as alterações do documento são exibidas com base no seu dispositivo.

<Tabs syncKey="platform">
    <TabItem label="Desktop">
  Na visão Desktop, você terá no canto superior esquerdo a barra de arquivos \
  ![file_bar](barra_de_arquivo_de_documentos_view_de_arquivos. ng)\
  Guarda, da esquerda para a direita, a <img alt="logo" src={logo.src} className="inline-icon" width="16"/> botão de borboleta para abrir o menu. Ao lado dele há o campo nome que exibe o nome da nota. You can rename the note by typing a new name into that field and saving it using the <FloppyDisk className="inline-icon"/> save button. Finally you can <MagnifyingGlass className="inline-icon"/> search for elements within the note.
  \
  At the top right of the screen you have the toolbar\
  ![toolbar](document_view_toolbar.png)\
  By default this bar has the <ScribbleLoop className="inline-icon"/> lasso tool; the <Pen className="inline-icon"/> pen tool; the <Path className="inline-icon"/> path erase tool; the <ArrowCounterClockwise className="inline-icon"/> undo and <ArrowClockwise className="inline-icon"/> redo butons, and the <Hand className="inline-icon"/> hand tool. Então você tem o botão <Plus className="inline-icon"/> para adicionar elementos para a barra de ferramenta, o botão <Wrench className="inline-icon"/> para configurar o documento, e, finalmente, a ferramenta <Lock className="inline-icon"/> para olhar o zoom e/ou a posição da tela.
  ### Notas importantes
  1. Se algum dos elementos na barra de ferramentas não estiverem visíveis para você, clique e arraste (ou deslize) para esquerda e direita para revelar ferramentas adicionais.
  2. Após selecionar a ferramenta, aparecerá um pequeno menu. Toque longo na ferramenta novamente lhe permitirá arrastá-la para uma nova posição
  3. Você pode adicionar mais ferramentas na barra de ferramentas clicando no símbolo <Plus className="inline-icon"/> mais.
 </TabItem>
