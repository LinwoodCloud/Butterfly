Tämä sivu kattaa sivujärjestelmän, joka erottelee eri muistiinpanoja yhdessä asiakirjassa. Jos etsit tapaa rajoittaa ääretöntä kangasta, ole hyvä ja [käytä alueen ominaisuutta](../area)

:::
