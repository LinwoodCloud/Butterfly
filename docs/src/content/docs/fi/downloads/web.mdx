## Versiot

Butterfly pää- ja esikatselu (kehittää) versiot eivät ole samat kuin vakaat ja yölliset versiot.
Ne eivät liity suoraan vapautukseen. Niitä päivitetään heti, kun uusi toimitus on työnnetty arkistoon.
