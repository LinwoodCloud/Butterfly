## Windows SmartScreen

Windows varoittaa käyttäjiä asennettaessa ohjelmistoa ilman varmennetta.

![Älykäs näyttö](@assets/img/smart-screen.png)

Voit asentaa sovelluksen, sinun täytyy klikata "More info".

![Älykäs näyttö lisätieto](@assets/img/smart-screen-more-info.png)

Klikkaa sitten "Suorita joka tapauksessa".
