---
title: Eraser tool
---

With this tool you can erase the elements on the paper.

## Konfiguracja

|         Własność | Domyślny | Opis                                                               |
| ---------------: | :------: | :----------------------------------------------------------------- |
| Szerokość obrysu |     5    | Szerokość skoku                                                    |
|  Mnożnik uderzeń |     1    | Jeśli masz styl, to jest mnożnik, który zostanie dodany do nacisku |
