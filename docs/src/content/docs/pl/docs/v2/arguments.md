---
title: Argumenty
---

Możesz uruchomić aplikację argumentami.

`butterfly.exe <Path>`

Ścieżka jest ścieżką do pliku, który chcesz otworzyć.

## Opcje

|          Opcja | Opis                                                                                     |
| -------------: | :--------------------------------------------------------------------------------------- |
| `--path`, `-p` | Wybierz katalog danych, w którym przechowywane są wszystkie dane Motyla. |
