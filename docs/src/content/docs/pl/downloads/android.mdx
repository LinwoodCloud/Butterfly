:::

## Program beta w sklepie Play

Kliknij [ten link](https://play.google.com/apps/testing/dev.linwood.butterfly), aby zarejestrować wersję [nightly](/community/nightly) aplikacji. Pamiętaj, że ta wersja może mieć błędy. Wersja sklepu odtwarzania zostanie automatycznie zaktualizowana po wydaniu nowej wersji nocnej.
