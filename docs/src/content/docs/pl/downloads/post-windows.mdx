## SmartScreen Windows

Windows ostrzega użytkowników podczas instalacji oprogramowania bez certyfikatu.

![Inteligentny ekran](@assets/img/smart-screen.png)

Aby zainstalować aplikację, musisz kliknąć "Więcej informacji".

![Inteligentny ekran więcej informacji](@assets/img/smart-screen-more-info.png)

Następnie kliknij "Uruchom mimo to".
