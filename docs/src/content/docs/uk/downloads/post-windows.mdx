## SmartScreen Windows

Windows попереджає користувачів під час встановлення програмного забезпечення без сертифікату.

![Smart screen](@assets/img/smart-screen.png)

Щоб встановити додаток, необхідно натиснути кнопку "Більше інформації".

![Смарт-екран детальніше](@assets/img/smart-screen more-info.png)

Потім натисніть "Запустити в будь-якому випадку".
