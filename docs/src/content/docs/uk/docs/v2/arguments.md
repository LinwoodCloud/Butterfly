---
title: Аргументи
---

Можете починати застосунок з аргументами.

`butterfly.exe <Path>`

Шлях до файлу, який потрібно відкрити.

## Опції

|          Опція | Опис                                                                       |
| -------------: | :------------------------------------------------------------------------- |
| `--path`, `-p` | Виберіть каталог даних для зберігання всіх даних метелика. |
