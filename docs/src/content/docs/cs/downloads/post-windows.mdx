## Windows SmartScreen

Windows upozorňuje uživatele při instalaci softwaru bez certifikátu.

![Smart screen](@assets/img/smart-screen.png)

Chcete-li nainstalovat aplikaci, musíte kliknout na "Další informace".

![Inteligentní obrazovka více informací](@assets/img/smart-screen-more-info.png)

Pak klikněte na "Spustit tak jako také".
