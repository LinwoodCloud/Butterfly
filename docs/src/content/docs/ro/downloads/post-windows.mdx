## Windows SmartScreen

Windows avertizează utilizatorii când instalează software-ul fără certificat.

![Ecran inteligent](@assets/img/smart-screen.png)

Pentru a instala aplicația, trebuie să faceți clic pe "Mai multe informații".

![Ecran inteligent mai multe informații](@assets/img/smart-screen-more-info.png)

Apoi, fă clic pe "Rulează oricum".
