## Einführung

![Area](area.png)

Flächen sind eine Möglichkeit, die Größe der Leinwand zu beschränken. Du kannst Flächen erstellen, indem du das [Area Tool](../tools/area). Wenn Sie auf den Hintergrund klicken, können Sie den Bereichslisten-Dialog über das Kontextmenü öffnen.

## Bereiche erstellen und löschen

Während Sie den Inhalt direkt auf der Leinwand bearbeiten können, es ist üblich, dass Sie die Region einschränken möchten, in der Sie den Inhalt ändern können oder dass Sie schnell auf einen Teil des Dokuments zugreifen können. Dafür haben Sie das <Monitor className="inline-icon"/> Flächenwerkzeug. Wie im [Einführung](../) erklärt, musst du möglicherweise den <Monitor className="inline-icon"/> Bereichs-Button zu deiner Symbolleiste hinzufügen. Mit dem [Flächen-Tool](../tools/area) kannst du Rechteck irgendwo auf der Leinwand zeichnen, um dann den Bereich zu definieren.

## Mehr Kontrolle

Sie können das Flächenwerkzeug weiter definieren, indem Sie erneut auf die Schaltfläche <Monitor className="inline-icon"/> klicken. Dort können Sie die feste Breite, Höhe oder das Seitenverhältnis der Bereiche festlegen.

## Das Kontextmenü des Bereichs

 Nach dem Zeichnen eines Bereichs können Sie auf den Bereich klicken oder lange tippen, um ein Kontextmenü mit den folgenden Optionen zu öffnen:

- `Name` - Gib dem Bereich einen Namen, der für dich überzeugend ist.
- `Betrete den Bereich` - Erlaube dir, Inhalte zu bearbeiten, die nur von diesem Bereich begrenzt sind.
- `Löschen` - Lösche den Bereich. _Löscht den Inhalt in diesem Bereich nicht_
- `Export` - Exportiere den Inhalt des ausgewählten Bereichs.
- `Zu Paket hinzufügen` - Füge den Bereich zu einem [pack](../pack) hinzu

## Eine andere Option

Eine weitere Option, mit der Sie den Bereichsnavigator verwenden können, ist im linken Menü, oder auf dem Schmetterlingsmenü, wenn Sie eine mobile Plattform verwenden.
