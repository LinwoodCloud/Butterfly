---
title: Argumente
---

Sie können die App mit Argumenten starten.

`butterfly.exe <Path>`

Pfad ist der Pfad zu der Datei, die Sie öffnen möchten.

## Optionen

|         Option | Beschreibung                                                                                       |
| -------------: | :------------------------------------------------------------------------------------------------- |
| `--path`, `-p` | Wählen Sie das Datenverzeichnis, in dem alle Daten von Butterfly gespeichert sind. |
