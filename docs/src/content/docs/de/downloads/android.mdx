## Das Beta-Programm im Play Store

Klicke auf [diesen Link](https://play.google.com/apps/testing/dev.linwood.butterfly) um dich für die [nightly](/community/nightly) Version der App zu registrieren. Bitte beachten Sie, dass diese Version Bugs haben kann. Die Play Store-Version wird automatisch aktualisiert, wenn eine neue nächtliche Version veröffentlicht wird.
