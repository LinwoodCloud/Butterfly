To edit the background of the page, click on the <Wrench className="inline-icon"/> icon and there, on the <Book className="inline-icon"/> page icon. Там у вас будет возможность нажать на значок " <Image className="inline-icon"/>". На компьютере можно также использовать клавишу `ctrl-B`.

При настройке backgourned, у вас есть две вкладки. Вкладка <Globe className="inline-icon"/> Общая для выбора типа используемого фона и <Gear className="inline-icon"/> для настройки выбранного фона.

## <Globe className="inline-icon"/> Pattern

Существует 8 различных фоновых шаблонов, которые вы можете выбрать. Для каждого из доступных шаблонов можно настроить следующее:

### <Gear className="inline-icon"/> Свойства

В горизонтальной вкладке конфигурации у вас есть следующий параметр:

- Ширина - ширина для каждой строки соответственно.
- Количество - так можно группировать строки в группу.
- Пространство - расстояние между каждой группой строк.

Например, для создания музыкального листа, вы можете использовать следующую конфигурацию:
`Ширина - 40`, `Count - 5`, `Space - 80`

Вертикальные настройки следуют той же логике для колонок вместо строк

## <Globe className="inline-icon"/> изображение

Тип фона изображения позволяет загрузить изображение и использовать его в качестве фона страницы.

### <Gear className="inline-icon"/> Свойства

Масштаб в направлении x и y можно изменить, если это изображение будет малым или большим.
