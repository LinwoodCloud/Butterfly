## Смартэкран Windows

Windows предупреждает пользователей при установке программного обеспечения без сертификата.

![Умный экран](@assets/img/smart-screen.png)

Чтобы установить приложение, нажмите на кнопку "Подробнее".

![Больше умного экрана](@assets/img/smart-screen-more-info.png)

Затем нажмите на кнопку «Все равно выполнить».
