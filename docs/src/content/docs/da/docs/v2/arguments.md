---
title: Argumenter
---

Du kan starte appen med argumenter.

`butterfly.exe <Path>`

Sti er stien til den fil du vil åbne.

## Indstillinger

|   Valgmulighed | Varebeskrivelse                                                     |
| -------------: | :------------------------------------------------------------------ |
| `--path`, `-p` | Vælg datamappe hvor alle data fra Butterfly gemmes. |
