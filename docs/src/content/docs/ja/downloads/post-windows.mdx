## Windows SmartScreen

Windowsは証明書なしでソフトウェアをインストールするときにユーザーに警告します。

![スマートスクリーン](@assets/img/smart-screen.png)

アプリをインストールするには、「詳細情報」をクリックする必要があります。

![スマートスクリーン情報](@assets/img/smart-screen-more-info.png)

次に、「とにかく実行」をクリックします。
