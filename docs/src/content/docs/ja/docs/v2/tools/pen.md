---
title: Pen tool
---

With this tool you can draw on this paper.

## 設定

|       属性 | デフォルト | 説明                              |
| -------: | :---: | :------------------------------ |
|        色 |  ブラック | 描画される色                          |
|  ストロークの幅 |   5   | ストロークの幅                         |
|  ストローク乗数 |   1   | スタイラスを持っている場合、これは加算される乗数です      |
| ズームに依存する | false | これにより、ズームレベルに基づいてストローク幅が変更されます。 |
