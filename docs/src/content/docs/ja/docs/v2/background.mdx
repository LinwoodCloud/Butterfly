import {Wrench,

} from "@phosphor-icons/react";

ページの背景を編集するには、 <Wrench className="inline-icon"/> アイコンをクリックし、 <Book className="inline-icon"/> ページアイコンをクリックします。 そこで、 <Image className="inline-icon"/> アイコンをクリックするオプションがあります。 コンピュータでは、`ctrl-B`ショートカットを使用することもできます。

backgourndを設定するときは、2つのタブがあります。 <Globe className="inline-icon"/> 一般タブで使用する背景の種類を選択し、 <Gear className="inline-icon"/> 選択した背景を設定します。

## <Globe className="inline-icon"/> パターン

選択できる背景テンプレートは8種類あります。 使用可能な各パターンについて、次のように構成できます。

### <Gear className="inline-icon"/> プロパティ

水平方向の構成タブでは、次のオプションがあります。

- 幅 - 各行の幅。
- カウント - その方法で行をグループ化できます。
- スペース - 行の各グループ間の距離。

例えば、音楽シートを作成するには、次の設定を使用できます。
`Width - 40`, `Count - 5`, `Space - 80`

垂直方向の構成は行の代わりに列の同じロジックに従っています

## <Globe className="inline-icon"/> 画像

画像の背景タイプでは、画像をアップロードしてページの背景として使用できます。

### <Gear className="inline-icon"/> プロパティ

この画像が小さい場合や大きい場合は、x 方向と y 方向のスケールを変更できます。
